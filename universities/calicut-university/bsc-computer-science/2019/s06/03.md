# BCS6B12: Operating Systems

## Course Objectives
* Learn the structure, functions, and concepts of operating systems.
* Explore process management, memory management, file systems, and I/O systems.
* Understand scheduling, synchronization, and deadlock handling.

## Course Outcomes
* **CO 1:** Explain the role and structure of operating systems.
* **CO 2:** Manage processes, scheduling, and synchronization using OS mechanisms.
* **CO 3:** Implement and analyze memory allocation and virtual memory techniques.
* **CO 4:** Understand file systems, storage management, and access control.
* **CO 5:** Evaluate I/O handling methods, disk scheduling, and RAID configurations.

## Course Content

### Module - 1 (Introduction)
* Operating system functions and types.
* System calls.
* System structure.

### Module - 2 (Process Management)
* Processes and threads.
* CPU scheduling: FCFS, SJF, Round Robin, Priority scheduling.
* Process synchronization: Semaphores, deadlock handling.

### Module - 3 (Memory Management)
* Contiguous and paging allocation.
* Virtual memory: Segmentation, demand paging.
* Page replacement algorithms.

### Module - 4 (File Systems)
* File concepts and access methods.
* Directory structures.
* File protection and allocation methods.

### Module - 5 (I/O and Storage)
* I/O hardware, polling, and interrupts.
* Disk scheduling algorithms.
* RAID levels.
